Instructions:
  • Double-click on a cell to edit. Note only "Modify Metric", "Sequence", and "Updated By" are editable. Edits are automatically saved to the database.
  • To attach a new Metric to a Dashboard, click Add Metric to Dashboard button. A new Metric row will appear that can be edited. Edits are automatically saved to the database.
  • To delete a Metric (s) from a Dashboard, first select a Metric Row(s) by checking the lefthand side checkmark and then click Delete Metric button.
