Instructions:
  • Double-click on a cell to edit. Note only "Dashboard", "Descriptions", and "Updated By" are editable. Edits are automatically saved to the database.
  • To create a new Dashboard, click Add Dashboard button. A new Dashboard row will appear that can be edited. Edits are automatically saved to the database.
  • To delete a Dashboard(s), first select a Dashboard Row(s) by checking the lefthand side checkmark and then click Delete Dashboard button.
